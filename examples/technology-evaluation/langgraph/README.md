# LangGraph Technology Evaluation Example

This directory contains a deterministic Technology Research & Evaluation Agent pipeline example for:

`Evaluate LangGraph for long-running AI agent workflows.`

Files:

- `input.md`: the user request.
- `evaluation_report_langgraph.json`: structured `EvaluationReport` payload.
- `technology_evaluation_report_langgraph.md`: Markdown rendered from the same payload.

This example demonstrates the local deterministic pipeline:

1. Criteria are scored with `evaluation_scorecard`.
2. The payload is preflight checked with `evaluation_report_validate`.
3. JSON and Markdown artifacts are generated by `evaluation_report_assembly`.
4. Markdown includes 13 fixed report sections plus Consistency Notes.

This is not a live network research run. The source entries are representative and should be refreshed by a live research pass before making a real adoption decision.
